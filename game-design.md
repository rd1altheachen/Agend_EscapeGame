# 🪄 魔法學徒的畢業考試 — 遊戲企劃方案 v2

## 一、故事背景

你是魔法學徒「小星」，今天是畢業考試日。
導師貓頭鷹魔法師「奧利弗教授」把你鎖在魔法工坊裡，留下一封信：

> 「親愛的小星，恭喜你來到最後的考驗！
> 解開工坊裡的所有魔法謎題，集齊六顆魔法星星，就能打開大門畢業啦。
> 提示：兩個房間的秘密是互相連結的喔～
> 祝你好運！🦉」

**目標：** 解開兩個房間共 12 個謎題，集齊 6 顆魔法星星（每個房間 3 顆），念出畢業咒語逃出。

---

## 二、房間設計

### 房間 1：星光書房 ⭐📚

溫暖的圓形書房，深藍色星空壁紙，金色點綴。

**場景物件：**
- 大書桌（有抽屜、魔法日記、羽毛筆）
- 書架（各色魔法書，書脊有符號）
- 魔法鏡（照出隱藏事物）
- 星座儀（可旋轉天球）
- 掛鐘（指針停住）
- 魔法門（通往房間2，一開始鎖住）
- 窗台小花盆（會發光的小花）
- 牆上星星拼圖框（最終收集用）
- 地毯（有隱藏圖案）
- 壁爐（裡面有東西）

---

### 房間 2：藥水廚房 🧪🍵

色彩繽紛的魔法廚房，暖橘+綠色調。

**場景物件：**
- 大鍋（煮藥水用）
- 食材架（各種魔法材料）
- 配方牆（貼著藥水配方，部分模糊）
- 音樂盒（可轉動，有彩色按鈕）
- 櫥櫃（上鎖，需密碼）
- 貓頭鷹掛畫（眼睛會動）
- 彩色瓶子排列架
- 天秤（可放東西稱重）
- 水晶球（可以窺視房間1的某個角落）
- 魔法門（回房間1）

## 三、謎題設計（共 12 題）

---

### 🔓 房間1 — 前 3 題（線索在房間1，解完開門進房間2）

#### 謎題 1-1：魔法鏡的秘密
- **觀察：** 點擊魔法鏡，鏡中映出書房，但鏡中書架有一本書在發光（現實看不到）
- **解法：** 點擊現實書架中對應位置的書
- **獲得：** 書中夾著「星座對照表」（顏色↔星座對應）
- **獲得星星：** ⭐ 第1顆星星

#### 謎題 1-2：魔法日記的暗號
- **觀察：** 翻開書桌上的魔法日記，每頁有插圖，其中 3 頁的插圖邊角有小小的符號（△、○、☆）
- **操作：** 注意日記中提到「按照月相順序排列符號」，日記中也畫了月相順序（新月→半月→滿月）對應（△→☆→○）
- **解法：** 在書桌抽屜的符號鎖上按 △☆○ 順序輸入
- **獲得：** 抽屜打開，裡面有「藥水配方殘頁」+「一把水晶鑰匙」
- **獲得星星：** ⭐ 第2顆星星

#### 謎題 1-3：開啟魔法門
- **操作：** 魔法門上有三個鑰匙孔形狀的凹槽（但只需要一把鑰匙+一個咒語）
- **線索：** 地毯上有隱藏圖案（掀開地毯角落可看到），圖案顯示：鑰匙插中間孔 + 對著門說出書架上某本書書脊的文字
- **解法：** 用水晶鑰匙插入中間凹槽 → 點擊書架上書脊寫著「開啟 Aperio」的那本書 → 門打開
- **獲得：** 魔法門開啟，可以進入房間2！
- **獲得星星：** ⭐ 第3顆星星

> ✅ 此時玩家可以自由在兩個房間間移動

---

### 🔒 房間1 — 後 3 題（線索在房間2，一開始解不了）

#### 謎題 1-4：星座儀密碼
- **問題：** 星座儀可旋轉，需要對準正確的 3 個星座才會打開
- **需要的線索（來自房間2）：** 房間2 音樂盒播放時彩色按鈕亮起順序（紅→藍→金）
- **解法：** 紅→藍→金 + 星座對照表（謎題1-1獲得）→ 獅子座、水瓶座、天秤座 → 旋轉對準
- **獲得：** 星座儀打開，裡面有「魔法砝碼」（房間2天秤需要用）+ ⭐ 第4顆星星

#### 謎題 1-5：壁爐暗格
- **問題：** 壁爐裡有個暗格，需要輸入 4 位數字密碼
- **需要的線索（來自房間2）：** 房間2 彩色瓶子排列架上，瓶子的數量按顏色分組（紅2、藍4、綠1、紫7）→ 密碼 2417
- **解法：** 輸入 2417
- **獲得：** 暗格打開，裡面有「月光粉」（房間2煮藥水需要）+ ⭐ 第5顆星星

#### 謎題 1-6：小花盆的秘密
- **問題：** 窗台小花盆的花一直閉合著，需要特定條件才會綻放
- **需要的線索（來自房間2）：** 房間2 煮出的「陽光藥水」（謎題2-5的產物）
- **解法：** 將陽光藥水倒在花盆上 → 花綻放，花心裡有最後一顆星星
- **獲得：** ⭐ 第6顆星星 → 集齊所有星星！

---

### 🔓 房間2 — 前 3 題（線索在房間2自己）

#### 謎題 2-1：音樂盒旋律
- **操作：** 轉動音樂盒把手，播放旋律，彩色按鈕按順序亮起（紅→藍→金）
- **觀察記住：** 這個顏色順序是房間1謎題1-4的線索
- **解謎本身：** 音樂盒底部有個小抽屜，需要按照旋律的節奏（長-短-短-長）按下按鈕才能打開
- **獲得：** 小抽屜打開，裡面有「攪拌匙」+ ⭐ 房間2第1顆星星

#### 謎題 2-2：貓頭鷹掛畫
- **觀察：** 貓頭鷹的兩隻眼睛可以分別點擊旋轉（像轉盤），每隻眼睛有4種圖案
- **線索：** 配方牆旁邊貼著一張小紙條：「貓頭鷹看見的是：左眼月亮，右眼星星」
- **解法：** 左眼轉到月亮圖案，右眼轉到星星圖案 → 貓頭鷹嘴巴打開
- **獲得：** 嘴巴裡吐出「彩虹露」（煮藥水材料）+ ⭐ 房間2第2顆星星

#### 謎題 2-3：彩色瓶子排列
- **觀察：** 食材架上有一排瓶子被打亂了，需要按正確順序排列
- **線索：** 配方牆上有一張褪色的圖，顯示瓶子應該按照彩虹色排列（紅橙黃綠藍靛紫）
- **解法：** 拖動瓶子按彩虹順序排好
- **注意：** 排好後，玩家可以數各顏色瓶子數量（紅2、藍4、綠1、紫7）→ 這是房間1謎題1-5的線索
- **獲得：** 排列正確後食材架底部彈出「星塵粉」+ ⭐ 房間2第3顆星星

---

### 🔒 房間2 — 後 3 題（線索在房間1）

#### 謎題 2-4：櫥櫃密碼鎖
- **問題：** 櫥櫃有 3 位數密碼鎖
- **需要的線索（來自房間1）：** 房間1 掛鐘停在 3:15 + 魔法日記寫著「時間倒流揭示真相」→ 3:15 倒轉 = 9:45 → 密碼 945
- **解法：** 輸入 945
- **獲得：** 櫥櫃打開，裡面有「魔法火種」（點燃大鍋用）

#### 謎題 2-5：煮藥水
- **問題：** 大鍋需要正確材料+正確順序才能煮出藥水
- **需要的線索（來自房間1）：** 謎題1-2獲得的「藥水配方殘頁」補全了配方牆上模糊的部分
- **完整配方：** 魔法火種點燃 → 彩虹露 → 星塵粉 → 月光粉 → 用攪拌匙攪拌 3 次
- **材料來源：** 火種（謎題2-4）、彩虹露（謎題2-2）、星塵粉（謎題2-3）、月光粉（房間1謎題1-5）、攪拌匙（謎題2-1）
- **解法：** 按順序放入材料並攪拌
- **獲得：** 煮出「陽光藥水」（帶回房間1解謎題1-6用）

#### 謎題 2-6：天秤之謎
- **問題：** 天秤一邊放著一個神秘寶石，另一邊需要放上等重的東西才會觸發機關
- **需要的線索（來自房間1）：** 謎題1-4獲得的「魔法砝碼」
- **解法：** 將魔法砝碼放上天秤 → 平衡 → 機關觸發，天秤底座打開
- **獲得：** 底座裡有「畢業咒語卷軸」

## 四、完整遊戲流程

```
【第一階段：房間1 前3題】
1-1 魔法鏡 → 星座對照表 + ⭐
1-2 魔法日記暗號 → 配方殘頁 + 水晶鑰匙 + ⭐
1-3 開啟魔法門 → 進入房間2 + ⭐

【第二階段：房間2 前3題】
2-1 音樂盒 → 攪拌匙 + ⭐（同時記住顏色順序給1-4）
2-2 貓頭鷹掛畫 → 彩虹露 + ⭐
2-3 彩色瓶子排列 → 星塵粉 + ⭐（同時記住數量給1-5）

【第三階段：交叉解謎】
1-4 星座儀（用2-1顏色+1-1對照表）→ 魔法砝碼 + ⭐
1-5 壁爐暗格（用2-3瓶子數量）→ 月光粉 + ⭐
2-4 櫥櫃密碼（用房間1掛鐘倒轉）→ 魔法火種
2-5 煮藥水（用1-2配方+各材料）→ 陽光藥水
2-6 天秤（用1-4魔法砝碼）→ 畢業咒語卷軸

【最終階段】
1-6 小花盆（用2-5陽光藥水）→ ⭐ 第6顆
→ 集齊6星 + 咒語卷軸 → 對大門使用 → 通關！🎉
```

## 六、操作方式（UI/UX）

### 基本操作
- **點擊物件：** 查看/互動
- **物品欄：** 畫面底部，收集到的物品顯示在這裡
- **使用物品：** 先點物品欄選中，再點場景目標
- **切換房間：** 點擊魔法門（開門後才能切換）
- **查看物品：** 雙擊物品欄物品可放大查看細節

### 畫面配置
```
┌─────────────────────────────────┐
│         場景畫面區域              │
│    （可點擊互動的物件）           │
│                                 │
├─────────────────────────────────┤
│  [物品1] [物品2] [物品3] ...     │  ← 物品欄
├─────────────────────────────────┤
│  [房間1] [房間2]    [提示💡]     │  ← 導航列
└─────────────────────────────────┘
```

### 提示系統
- 點擊「提示💡」可獲得當前卡關謎題的漸進式提示（最多3層，越來越明確）

## 七、美術風格

- **整體：** 手繪水彩感、圓潤線條、柔和配色
- **房間1：** 深藍+金色（星空夜晚感）
- **房間2：** 暖橘+綠色（溫馨廚房感）
- **動畫：** 物件互動有彈跳/閃光效果
- **禁止：** 恐怖、血腥、獵奇、陰暗元素
